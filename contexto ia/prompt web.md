Replica la landing page mostrada en la captura adjunta (rootsbystella.es), 
una web para un salón de peluquería especializado en trenzas africanas y 
soluciones capilares en Menorca.

STACK Y ESTRUCTURA
- HTML5 semántico + CSS moderno (variables CSS, flexbox/grid) + JS vanilla mínimo
- Una sola página (single page) con scroll suave entre secciones
- Totalmente responsive (mobile-first, breakpoints en 768px y 1024px)
- Sin frameworks pesados; si necesitas algo, usa solo CSS puro

PALETA DE COLORES
- Negro principal: #0a0a0a (fondos hero y sección testimonios)

- Dorado: #d4b04a (logo y acentos tipográficos)
- Blanco: #ffffff

DETALLES IMPORTANTES
- Todos los títulos de sección llevan una pequeña línea dorada horizontal 
  decorativa encima (~40px de ancho)
- Botones con border-radius pequeño (~4-6px), no completamente redondeados
- Espaciado generoso entre secciones (padding vertical mínimo 80px desktop)
- Las imágenes usa placeholders de Unsplash relevantes (trenzas africanas, 
  salón de belleza, retratos) hasta que se sustituyan por las reales
- Menú móvil: hamburguesa que despliega overlay full-screen
- Suaviza con transiciones CSS los hover de botones y tarjetas
